# AiXel Program Dashboard

Platform monitoring dan manajemen program AiXel — AI-Accelerated Excellence Program oleh Talentlytica.

## Live Dashboard
🔗 **[https://ichbalmunasdar.github.io/aixel-program/](https://ichbalmunasdar.github.io/aixel-program/)**

## Fitur Dashboard
- **Participant Management** — Kelola data peserta program
- **Monthly Assessment** — Input & tracking penilaian bulanan
- **Community Log** — Catatan aktivitas komunitas
- **Quarterly Review** — Evaluasi performa per kuartal
- **Firebase Realtime Database** — Cloud storage terintegrasi

## Teknologi
- HTML/CSS/JavaScript (Single Page Application)
- Firebase Realtime Database (compat SDK v9.22.2)
- GitHub Pages (Hosting)
